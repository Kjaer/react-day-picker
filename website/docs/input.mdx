You may need to integrate DayPicker with an input field, to let user _type in a date_ instead of clicking the calendar.

This recipe shows how to add an input field that sync with the selected day in DayPicker.

### 1. Setup a DayPicker and an Input Field

Create a basic component with `input` and `DayPicker`:

```jsx
function App() {
  return (
    <form className="center">
      <label>
        <p>Type a day or pick one from the calendar.</p>
        <input />
      </label>
      <br />
      <DayPicker />
    </form>
  );
}
```

### 2. Define the Date Format

The input field should display the formatted day selected from DayPicker.

For example, set `yyyy-MM-dd` using date-fns’s [format](https://date-fns.org/v2.4.1/docs/format) function and add a placeholder to the input.

```jsx
function App() {
  const FORMAT = 'yyyy-MM-dd';
  return (
    <form className="center">
      <label>
        <p>Type a day or pick one from the calendar.</p>
        <input placeholder="YYYY-MM-DD" />
      </label>
      <br />
      <DayPicker />
    </form>
  );
}
```

### 3. Set the Input Field Value when a Day is Clicked

Control the `input` field with an internal state, and set its formatted value when a day is clicked in the calendar.

```jsx
import React, { useState } from 'react';
import { isSameDay } from 'date-fns';

function App() {
  const [value, setValue] = useState();
  const FORMAT = 'yyyy-MM-dd';

  // Update the input field
  const handleDayClick = day => setValue(format(day, FORMAT));

  return (
    <form className="center">
      <label>
        <p>Type a day or pick one from the calendar.</p>
        <input value={value} placeholder="YYYY-MM-DD" />
      </label>
      <br />
      <DayPicker onDayClick={handleDayClick} value={value} />
    </form>
  );
}
```

_Select a day to see its formatted value in the input field_.

### 4. Parse the Typed Value

Typing a valid date into the field should update the selected day in the calendar, too. Here we can add some requirements:

- if it is a valid date, set the date as selected in DayPicker;
- if the text typed is not a valid date, do not select anything.

```jsx preview previeweight=400
import React, { useState } from 'react';
import { DayPicker } from 'react-day-picker';
import { format, parse } from 'date-fns';

function App() {
  const [selected, setSelected] = useState();
  const [value, setValue] = useState();

  const FORMAT = 'yyyy-MM-dd';
  const handleChange = e => {
    setValue(e.target.value);

    // Set the selected day, if the typed value is a valid date.
    const day = parse(e.target.value, FORMAT, new Date());

    if (isNaN(day.getTime())) {
      // Invalid date, unselect the day
      setSelected(undefined);
      return;
    }

    setSelected(day);
  };

  const handleDayClick = (day, modifiers) => {
    if (modifiers.selected) {
      // Empty selected day in case clicking the same day
      setSelected();
      setValue('');
      return;
    }
    setSelected(day);
    setValue(format(day, FORMAT));
  };

  return (
    <form className="center">
      <label>
        <p>Type a day or pick one from the calendar.</p>
        <input placeholder="YYYY-MM-DD" value={value} onChange={handleChange} />
      </label>
      <br />
      <DayPicker selected={selected} onDayClick={handleDayClick} />
    </form>
  );
}
```

_Try playing with input and clicking the days._

### 5. Update the Month when Typing the Day

If the typed day is outside the current month, it won't be visible: we need to control the month shown in the calendar.

To update the month in DayPicker when typing the day:

1. add the current month to the component’s state;
2. use the `month` and the `onMonthChange` props to update the current onMonthChange;
3. set the current month to the one typed in the field.

```jsx preview previewHeight=400
import React, { useState } from 'react';
import { DayPicker } from 'react-day-picker';
import { format, parse } from 'date-fns';

function App() {
  const [selected, setSelected] = useState();
  const [value, setValue] = useState();

  // Save the current month in state
  const [currentMonth, setCurrentMonth] = useState(new Date());

  const FORMAT = 'yyyy-MM-dd';
  const handleChange = e => {
    setValue(e.target.value);
    const day = parse(e.target.value, FORMAT, new Date());
    if (isNaN(day.getTime())) {
      setSelected(undefined);
      return;
    }
    setSelected(day);

    // Update the month shown in the calendar.
    setCurrentMonth(day);
  };

  const handleDayClick = (day, modifiers) => {
    if (modifiers.selected) {
      setSelected();
      setValue('');
      return;
    }
    setSelected(day);
    setValue(format(day, FORMAT));
  };

  return (
    <form className="center">
      <label>
        <p>Type a day or pick one from the calendar.</p>
        <input placeholder="YYYY-MM-DD" value={value} onChange={handleChange} />
      </label>
      <br />
      <DayPicker
        month={currentMonth}
        onMonthChange={setCurrentMonth}
        selected={selected}
        onDayClick={handleDayClick}
      />
    </form>
  );
}
```

Other features that are easy to add:

- highlight the field when the date is not valid;
- reset the original date when the date is not valid and the field loses focus;
- place DayPicker in a pop-over, for example with [popper.js](https://popper.js.org).

...have fun cooking with DayPicker!
